Description : lst20-cls (LST20)

Long Description : CLS from LST20 dataset. LST20 corpus are free for research and open source only. If you want to use in Commercial use, please contract NECTEC.

Download and Use

Download

from pythainlp.corpus import download
download('lst20-cls')

Use

It's get path file of corpus.
from pythainlp.corpus import get_corpus_path
get_corpus_path('lst20-cls')
if get_corpus_path('lst20-cls') is None than you not download lst20-cls.
