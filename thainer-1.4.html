Download and Use

Download

download('thainer-1.4')

Use

It's get path file of corpus.

from pythainlp.corpus import get_corpus_path
get_corpus_path('thainer-1.4')

If get_corpus_path('thainer-1.4') is None then you have not downloaded thainer-1.4.
