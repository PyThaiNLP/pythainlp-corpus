tnc_trigram_word_freqs

latest version: 2017


Description: Trigram word frequency from Thai National Corpus (TNC)

Long Description: Trigram word frequency from Thai National Corpus (TNC)

HomePage: http://www.arts.chula.ac.th/ling

Download and Use

Download

from pythainlp.corpus import download
download('tnc_trigram_word_freqs')

Use

It's get path file of corpus.

from pythainlp.corpus import get_corpus_path
get_corpus_path('tnc_trigram_word_freqs')

If get_corpus_path('tnc_trigram_word_freqs') is None then you have not downloaded tnc_trigram_word_freqs.
