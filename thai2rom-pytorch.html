Download and Use

Download

from pythainlp.corpus import download
download('thai2rom-pytorch')

Use

It's get path file of corpus.
from pythainlp.corpus import get_corpus_path
get_corpus_path('thai2rom-pytorch')
if get_corpus_path('thai2rom-pytorch') is None than you not download thai2rom-pytorch.
